## 📖 Descripción  

Este proyecto consiste en el desarrollo de un portafolio web personal utilizando Bootstrap, donde se presentan datos personales, habilidades, proyectos y un medio de contacto en una interfaz moderna y responsiva.


Se tomó como referencia la siguiente plantilla:

🔗 https://startbootstrap.com/previews/personal

### 📌 Secciones del portafolio

- **Inicio:**  
  Muestra la información principal del usuario, como nombre, imagen de perfil y botones de navegación.

- **Sobre mí:**  
  Contiene una breve descripción personal y académica.

- **Skills (Habilidades):**  
  Lista de tecnologías y conocimientos del usuario.

- **Proyectos:**  
  Sección donde se muestran proyectos mediante tarjetas con imagen y descripción.

- **Contacto:**  
  Información de contacto (correo electrónico).

---

## 🛠️ Proceso de creación

1. Se seleccionó una plantilla de portafolio basada en Bootstrap como referencia.
2. Se creó la estructura base en HTML (navbar, secciones, contenedores).
3. Se agregaron estilos personalizados en CSS con temática oscura/cyberpunk.
4. Se incorporaron tarjetas para mostrar proyectos.
5. Se añadieron botones con JavaScript para navegación con scroll suave.
6. Se simplificó la plantilla original eliminando elementos innecesarios.
7. Se personalizó el contenido con información del autor.
